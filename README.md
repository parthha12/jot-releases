# Jot — macOS downloads

Public release artifacts for **[Jot](https://parthha12.github.io/)** (macOS).

- **Download:** [Latest release](https://github.com/parthha12/jot-releases/releases/latest)
- **Website:** https://parthha12.github.io/
- **Source code:** private — this repo contains installers only.

## Install

1. Download `Jot-<version>.dmg` from [Releases](https://github.com/parthha12/jot-releases/releases/latest)
2. Drag **Jot** to Applications
3. If macOS blocks the app: `xattr -cr "/Applications/Jot.app"`
