| **Jot Workspace** | Full platform (crew, agents, recordings) | Private / [story](https://parthha12.github.io/workspace.html) |

## Install

1. Download `JOT-Reminders-*.dmg` from [Releases](https://github.com/parthha12/jot-releases/releases/latest)
2. Drag **J.O.T. Reminders** to Applications
3. If macOS blocks the app: `xattr -cr "/Applications/J.O.T. Reminders.app"`
